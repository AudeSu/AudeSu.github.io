Wat voor soort waarde zit er in die variabelen?

De variabele sliders zal verwijzen naar een HTMLCollection van één element (de slider) en de variabele colorDemos zal verwijzen naar een NodeList van één element (het blokje).

Hoeveel elementen zitten er in elke verzameling?

Er zit 1 element in elke verzameling.

Kun je uit de HTML code afleiden waarom er dit precies zoveel zijn?

Ja, de HTML-code bevat slechts één element met de klasse "colorDemo" en één input-element met de klasse "slider".

Op welke index positie in de lijst staat dit element?

Het eerste element staat op indexpositie 0 in de lijst.

Heb je er trouwens op gelet dat het blokje (nog) niet rood is?

Het blokje is nog niet rood omdat de setup-functie nog niet is uitgevoerd.

Snap je nu waarom er een [0] achteraan colorDemos[0] staat?

De [0] achter colorDemos[0] staat omdat getElementsByClassName een NodeList teruggeeft, en we slechts één element willen selecteren, dus gebruiken we de index 0 om het eerste element te selecteren.
